Controls

• Use the toolbar to select your (fluid) input mode, size and amount.
• Fluid Sources can be removed by tapping on them.
• The boat can be driven with the on-screen joysticks.
• The camera can be moved with the on-screen joysticks.
• Performance varies per device.
